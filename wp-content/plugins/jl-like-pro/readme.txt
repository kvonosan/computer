== Description ==

Add simple social share buttons to your articles and categories. Your visitors will be able to easily share your content on the most popular social networks: VKontakte, Facebook, ok.ru, Google Plus, Twitter, LinkedIn, mail.ru, Pinterest.

This extension of the social buttons for popular CMS Joomla. Now social buttons are available for WordPress.

Demo site plugin [JL Like PRO for WordPress](http://wp.joomline.ru/2015/11/24/jl-like/ "JL Like PRO for WordPress").

Description plugin: [Russian](http://joomline.ru/wordpress/629-wllikepro.html "Russian") and [English](http://joomline.org/wordpress/395-wllikepro.html "English")

[Forum Support](http://joomline.net/forum/plugins-wordpress.html "Forum Support")

= Key features =

* There are no third-party scripts.
* Lightweight.
* Own algorithms counting references in social networks.
* The ability to disable multiple like.
* Autogenerate og tag.
* High click through.
* Output in the category of materials.
* Ability to specify a default image.
* You can use CSS to change the look of social buttons.
* Localization: Russian and English.
* Support open graph Facebook.
* Total counter likes.
* Support control tag og: description.
* Default picture for a tag og: image.

== Installation ==

1. Install plugin from WordPress Plugin Directory or upload `jl-like-pro` to the `/wp-content/plugins/` directory manually.
2. Activate plugin through the 'Plugins' menu in WordPress admin panel.

== Frequently Asked Questions ==

= How work social buttons? = 
Social buttons based on technology share. With the API, social capabilities, we get a number of mentions of a page in social networks.

To display on the shortcode you can use the tag [jllikepro_buttons]
